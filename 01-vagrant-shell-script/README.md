🧪 01-vagrant-shell-script - Ambiente de laboratório básico com Vagrant

🔹Provisionamento de VM Linux (Ubuntu 20.04) com os seguintes requisitos:
1 - Criação de Shell Script de Provisionamento para instalação de pacotes (vim, curl, telnet, etc) e criação de um usuário.
2 - Acesso via SSH (vagrant ssh)
3 - Testes iniciais:
*Verificar instalação dos pacotes (dpkg -l | grep nome)
*verificar criação do usuário (id nome)

🧰 Tecnologias utilizadas
Vagrant
VirtualBox
Linux
Shell Script

▶️ Como usar
Entre no projeto desejado:
01-vagrant-shell-script
vagrant up
vagrant ssh
