🧪 projeto01 - Customização e automatização de uma vm linux utilizando Vagrant. O lab prover uma vmbox com acesso via ip público e sincronização de pasta do host local com a vm criada.

🔹Provisionamento de VM Linux (Ubuntu 24.04)
Configurações: 
-RAM: 1GB 
-CPU: 1núcleo
-Placa de rede em modo Bridge com (configuração de rede pública
Criação e sincronização de pasta local do host para outra pasta da vm a ser criada) 
-Acesso via SSH
-Acesso via ip público e verificação da pasta (volume) sincronizada.

🧰 Tecnologias utilizadas
Vagrant
VirtualBox
Linux

▶️ Como usar
Entre no projeto desejado:
cd /vagrant-projetos/projeto01
vagrant up
vagrant ssh
