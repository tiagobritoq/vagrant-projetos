# 🚀 Vagrant Labs - DevOps

## 📌 Descrição

Repositório com laboratórios práticos utilizando **Vagrant** para simulação de ambientes DevOps.

Cada pasta representa um cenário diferente de estudo e prática de infraestrutura como código.

---

## 📂 Estrutura do projeto

```bash
vagrant-projetos/
├── projeto01/
├── projeto02/
├── projeto03/
```

---

## 🧪 Projetos disponíveis

### 🔹 projeto01 - Ambiente básico com Vagrant

* Provisionamento de VM Linux
* Acesso via SSH
* Testes iniciais de automação

*(adicione mais conforme criar novos labs)*

---

## 🧰 Tecnologias utilizadas

* Vagrant
* VirtualBox
* Linux
* Shell Script

---

## ▶️ Como usar

Entre no projeto desejado:

```bash
cd projeto01
vagrant up
vagrant ssh
```
