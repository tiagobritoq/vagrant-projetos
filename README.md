# 🚀 Vagrant Labs - DevOps

## 📌 Descrição

Repositório com laboratórios práticos utilizando **Vagrant** para simulação de ambientes DevOps.

Cada pasta representa um cenário diferente de estudo e prática de infraestrutura como código.

---

## 📂 Estrutura do projeto

```bash
vagrant-projetos/
├── 00-vagrant-vmbox-custom/
├── 01-vagrant-shell-script/
├── 02-.../
```

## 🧪 Projetos disponíveis

### 🔹 00-vagrant-vmbox-custom

* Provisionamento de VM Linux Personalizda
* Sincronização de pasta entre host e vm criada
* Acesso via SSH
* Testes iniciais de automação

### 🔹 01-vagrant-shell-script 

* Provisionamento de VM Linux
* Instalação de lista de pacotes
* Criação de usuário local
* Acesso via SSH
* Testes iniciais de automação
---

## 🧰 Tecnologias utilizadas

* Vagrant
* VirtualBox
* Linux
* Shell Script

---

## ▶️ Como usar

Entre no projeto desejado:

```bash
cd 00-vagrant-vmbox-custom
vagrant up
vagrant ssh
```
